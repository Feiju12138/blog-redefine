【踩坑】Scp远程复制文件时报错

  • Linux 

前言

Scp远程复制文件时报错：scp: Received message too long scp: Ensure the remote shell produces no output for non-interactive sessions.

原因

  • 远端的环境变量中包含了输出语句

解决问题

  • 将远端的环境变量中可能会输出内容的命令去除
